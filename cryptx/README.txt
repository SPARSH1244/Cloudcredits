# 🔐 CryptX

CryptX is an advanced, GUI-based text and file encryption software built in Python using PyQt6. It supports AES, RSA, Fernet, and Caesar ciphers with drag-and-drop functionality, dark theme GUI, and future AI password strength feedback.

## 🔧 Features
- Multi-algorithm support (AES, RSA, Fernet, Caesar)
- Text and file encryption/decryption
- Drag & drop file support
- Local key vault UI (future integration)
- `.exe` and Linux-compatible
- Secure, offline operation
